# CSV Analyzer

A simple and interactive CSV Analyzer built with Streamlit and Pandas. Upload a CSV file and instantly explore your dataset with useful insights and previews.

## Features

* Upload CSV files
* View dataset shape (rows and columns)
* Preview the first few records
* Display column names
* View data types of each column
* Generate basic dataset statistics
* User-friendly web interface

## Tech Stack

* Python
* Streamlit
* Pandas

## Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/csv-analyzer.git
cd csv-analyzer
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the application:

```bash
streamlit run main.py
```
